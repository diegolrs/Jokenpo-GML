# RPG-Jokenpo
A RPG using Jokenpo in turns

## Ideias
 * ### Sistema de upgrade pelo tempo jogado
 * ### Personagens de tipos diferentes
 
 ## Habilidades únicas
 * ### Se curar
 * ### Mais dano
 * ### Mais vida
 * ### Mais resistência
 * ### Impedir dano ao dar empate
 * ### Velocidade de ataque (ataca antes)
